== Description ==

When a visitor coming from search engine, this plugin will retrieve query and insert it as tags of the post which the visitor landed on.<br />
Feel free to visit author's website <a href="http://www.ainging.com">Ainging</a> to report bugs.

== Installation ==

1. Extract insert-query-as-tags.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Insert Query As Tag" In Setting
4. Follow the on-screen instructions

== Frequently Asked Questions ==
= Children categories don't show up when I created new parent and children categories at the same time =
For people who use wordpress 3.0 or 3.0.1, please look at this ticket http://core.trac.wordpress.org/ticket/14485.
The temporary solution is to create another category or delete one category to refresh the wordpress built-in cache.
